# NBC_Quest4
1. 기존 코드 분석하기
    - [ ]  어떤 의도로 이 코드를 구현했을까?
          : 구조화를 통해 이름과 재료를 묶어 관리한다.
    - [ ]  수정을 해야 한다면, 기존 코드의 수정을 최소화 할 수 있는 방안이 있을까?
          : 기존의 코드를 캡슐화하고, call by value를 call by reference로 변경한다.
    - [ ]  기존 코드는 SOLID 원칙을 잘 준수하고 있나?
          : 미흡한 점이 많다. AlchemyWorkshop 클래스에서 수많은 로직을 담당하고 있고, 메인에선 이에 직접 의존하고 있다.

    - [ ]  물약 이름과 물약에 사용되는 재료 이름으로 창고를 검색하는 기능을 만드세요.
        - 예) 물약 ‘간장계란밥’을 검색할 때, 이름인 ‘간장계란밥’을 입력해도, ‘간장’, ‘계란’,’참기름’ 등 재료를 입력해도 검색이 되어야 합니다.
